# 备注

## 生僻字注音
对于生僻字采取悬浮注释与注音.其模版如下:

```html
/* ===== 悬停注释样式 ===== */
.hover-note {
  position: relative;
  cursor: help;
  border-bottom: 1px dashed #999;
}

.hover-note .note-text {
  visibility: hidden;
  width: max-content;
  max-width: 260px;
  background-color: #333;
  color: #fff;
  text-align: left;
  padding: 6px 10px;
  border-radius: 4px;
  font-size: 0.85em;

  position: absolute;
  z-index: 999;
  bottom: 125%;
  left: 50%;
  transform: translateX(-50%);
  opacity: 0;
  transition: opacity 0.2s;
}

.hover-note:hover .note-text {
  visibility: visible;
  opacity: 1;
}
<span class="hover-note">
  <ruby>晷<rt>guǐ</rt></ruby>
  <span class="note-text">晷：日影，这里指时间。</span>
</span>
```
效果如:

## 标注人名地名作品名

```html
<style>
  /* 人名：细实线，深灰 */
  .name {
    border-bottom: 1px solid #333;
    padding-bottom: 1px;
  }

  /* 地名：虚线，中灰 */
  .toponym {
    border-bottom: 1px dashed #666;
    padding-bottom: 1px;
  }

  /* 书名：稍粗实线，暗红（不用波浪线） */
  .book {
    border-bottom: 2px solid #b30000;
    padding-bottom: 2px;
  }
</style>
<span class="name">班固</span>在<span class="toponym">洛阳</span>创作了<span class="book">《两都赋》</span>，对比了<span class="toponym">长安</span>与<span class="toponym">洛阳</span>的盛况。
```
