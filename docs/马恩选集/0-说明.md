# 马恩选集卷一

![alt text](image-1.png)

## 说明

### 第一卷说明

第一卷主要收录马恩 {==1843-1859==}期间的著作以及后来为这些著作撰写的序言和导言. 

{==黑格尔法哲学批判导言==}中,马克思从唯物主义和无神论的立场出发,阐明了宗教的社会根源和本质,揭示了德国封建专制制度的社会基础和阶级制度,提出了推翻这一反动制度的战斗任务,论述了革命理论同革命实践相统一的思想,首次明确的阐述了无产阶级消灭一切奴役,实现人的解法的历史使命.
