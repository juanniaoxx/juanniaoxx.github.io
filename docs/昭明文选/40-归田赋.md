# 归田赋

## 作者与背景

张衡(78-139年),字平子,东汉南阳西鄂人(今河南南阳)人.浑天仪的发明者,汉赋四大家之一. 本赋做于东汉汉顺帝永和三年(138年),时年61岁的张衡在河间相任上,当时幼主暗弱宦官独揽王权,朝廷腐败,纲纪败坏,张衡感怀自己的政治抱负无法实现,故而做此短赋表达归田隐居的向往.
