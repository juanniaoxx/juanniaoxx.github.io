# 伯夷列传

![](../images/伯夷列传.png)

## 正文
夫学者载籍极博。尤考信于
<span class="hover-note">
  六艺
  <span class="note-text">指儒家六经:《诗》《书》《礼》《乐》《易》《春秋》</span>
</span>。《诗》、《书》虽缺[^1]，
然虞、夏之文可知也。尧将逊位，让于虞舜，舜、禹之间，岳牧咸荐，乃试之于位，典职数十年，功用既兴，然后授政。示天下重器，王者大统，传天下若斯之难也。而说者曰：“尧让天下于许由，许由不受，耻之逃隐[^2]。及夏之时，有卞随、务光者。”此何以称焉？
太史公曰：余登箕山，其上盖有许由冢云。孔子序列古之仁圣贤人，如吴太伯、伯夷之伦详矣。余以所闻，由、光义至高，其文辞不少概见，何哉？

孔子曰：“伯夷、叔齐，不念旧恶，怨是用希。”“求仁得仁，又何怨乎？”余悲伯夷之意，睹<span class="hover-note">
  轶诗
  <span class="note-text">指采薇,由于不编入诗三百故曰轶诗</span>
</span>。可异焉[^3]。其传曰：伯夷、叔齐，孤竹君之二子也。父欲立叔齐。及父卒，叔齐让伯夷。伯夷曰：“父命也。”遂逃去。叔齐亦不肯立而逃之。国人立其中子。于是伯夷、叔齐闻西伯昌善养老，“<ruby>盍<rt>hé</rt></ruby>往归焉！”及至，西伯卒，武王<span class="hover-note">
  载木主
  <span class="note-text">木主:祖宗灵位,这里指武王伐纣的时候携带文王的灵位</span>
</span>，号为文王，东伐纣。伯夷、叔齐叩马而谏曰：“父死不葬，<span class="hover-note">
  <ruby>爰<rt>yuán</rt></ruby>
  <span class="note-text">爰：乃,于是.</span>
</span>及干戈，可谓孝乎？以臣弑君，可谓仁乎？”左右欲兵之。太公曰：“此义人也。”扶而去之。武王已平殷乱，
天下宗周，而伯夷、叔齐耻之，义不食周粟，隐于首阳山，采薇而食之。
及饿且死，作歌，其辞曰：“登彼西山兮，采其薇矣。以暴易暴兮，不知其非矣。神农、虞、夏忽焉没兮，我安适归矣？于嗟<ruby>徂<rt>cú</rt></ruby>兮，命之衰矣。”遂饿死于首阳山。由此观之，怨邪非邪？ 

或曰：“天道无亲，常与善人。”若伯夷、叔齐，可谓善人者非邪？积仁洁行，如此而饿死。且七十子之徒，仲尼独荐颜渊为好学。然回也屡空，糟糠不厌，而卒蚤夭。天之报施善人，其何如哉？<span class="hover-note">
  盗<ruby>跖<rt>zhí</rt></ruby>
  <span class="note-text">春秋时期大盗,柳下惠之弟,见庄子.</span>
</span>[^4]日杀不辜，肝人之肉，{==暴戾恣睢==}，聚党数千人，横行天下，竟以寿终，是遵何德哉？此其尤大彰明较著者也。若至近世，操行不轨，专犯忌讳，而终身逸乐，富厚累世不绝。或择地而蹈之，时然后出言，行不由径，非公正不发愤，而遇祸灾者，不可胜数也。余甚惑焉，倘所谓天道，是邪非邪？

子曰：“<span class="hover-note">
  道不同，不相为谋。
  <span class="note-text">卫灵公篇</span>
</span>”亦各从其志也。故曰：“<span class="hover-note">
  富而可求也，虽执鞭之士，吾亦为之。如不可求，从吾所好。
  <span class="note-text">述而篇</span>
</span>”“<span class="hover-note">
  岁寒，然后知松柏之后凋也。
  <span class="note-text">子罕篇</span>
</span>”举世混浊，清士乃见。岂以其重若彼，其轻若此哉？

“<span class="hover-note">
  君子疾没世而名不称焉。
  <span class="note-text">卫灵公篇;君子最担心的是，直到死去了，名声也不被世人称道。</span>
</span>”贾子曰：“<span class="hover-note">
  贪夫徇财，烈士徇名，夸者死权，众庶冯生。
  <span class="note-text">贾谊 <ruby>鵩<rt>fú</rt></ruby>鸟赋;贪婪的人为钱财而死，刚烈的人为名声而死，贪慕虚荣的人为权势而死，普通百姓则依靠、依恋生存。</span>
</span>”同明相照，同类相求。“<span class="hover-note">
  云从龙，风从虎，圣人作而万物睹。
  <span class="note-text">周易·乾卦·文言传;比喻同类的事物会相互感应、相互追随。当圣王（或英雄豪杰）兴起时，世间万物（或各类人才）都能清楚地看见他，并归附于他。</span>
</span>”
伯夷、叔齐虽贤，得夫子而名益彰；颜渊虽笃学，附骥尾而行益显。岩穴之士，趋舍有时，若此类名湮灭而不称，
悲夫。{==闾巷之人，欲砥行立名者，非附青云之士，恶能施于后世哉! ==}

## 读后感 
作为列传的开篇，太史公其实讲了一个非常朴素又残酷的道理："岩穴之士，名湮灭而不称，悲夫！"

写《伯夷列传》的目的，不仅仅是为了纪念伯夷，更是为了确立《史记》的使命——做后世的‘青云’。就像孔子当年成就了伯夷的名声一样，太史公要通过《史记》这支笔，让那些原本会被历史洪流淹没的‘闾巷之人’，也能‘附骥尾而行益显’。

所谓‘好风凭借力’，这阵风不是让我们飞黄腾达的东风，而是对抗遗忘的史笔。太史公是在告诉后人：如果你是一个有操守的人，请不要绝望，我会把你们写进书里，让你们活在后人的记忆中。”
