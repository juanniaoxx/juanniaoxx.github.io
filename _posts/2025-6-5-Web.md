# 前端快速入门笔记

## Part I HTML

### HTML基础结构
```html
<!DOCTYPE html>
<html>
<head>
    <meta charset="UTF-8">
    <title>页面标题</title>
</head>
<body>
    <!-- 页面内容 -->
</body>
</html>
```

### 常用HTML标签
- 文本标签：
  ```html
  <h1>到<h6> 标题
  <p>段落</p>
  <span>行内文本</span>
  <strong>加粗</strong>
  <em>斜体</em>
  <br>换行
  <hr>水平线
  ```
  
- 列表：
  ```html
  <ul>无序列表</ul>
  <ol>有序列表</ol>
  <li>列表项</li>
  <dl>定义列表</dl>
  <dt>定义术语</dt>
  <dd>定义描述</dd>
  ```

- 表格：
  ```html
  <table>
    <tr>表格行</tr>
    <th>表头单元格</th>
    <td>表格单元格</td>
  </table>
  ```

- 表单：
  ```html
  <form>
    <input type="text"> 文本框
    <input type="password"> 密码框
    <input type="checkbox"> 复选框
    <input type="radio"> 单选按钮
    <select><option>下拉选择</option></select>
    <textarea>多行文本</textarea>
    <button>按钮</button>
  </form>
  ```

- 多媒体：
  ```html
  <img src="url" alt="描述"> 图片
  <audio>音频</audio>
  <video>视频</video>
  ```

### 块元素 vs 行内元素
- **块元素**：独占一行，可设置宽高（div, p, h1-h6, ul, li等）
- **行内元素**：不独占一行，宽高由内容决定（span, a, strong, em等）

### HTML5语义化标签
```html
<header>页眉</header>
<nav>导航</nav>
<section>文档章节</section>
<article>独立内容</article>
<aside>侧边栏</aside>
<footer>页脚</footer>
```

## Part II CSS

### CSS引入方式
1. 内联样式：`<p style="color:red;">`
2. 内部样式表：`<style> p {color:red;} </style>`
3. 外部样式表：`<link rel="stylesheet" href="style.css">`

### 常用选择器
| 选择器       | 示例       | 描述                     |
| ------------ | ---------- | ------------------------ |
| 元素选择器   | `p`        | 选择所有<p>元素          |
| 类选择器     | `.class`   | 选择class="class"的元素  |
| ID选择器     | `#id`      | 选择id="id"的元素        |
| 后代选择器   | `div p`    | 选择div内的所有p元素     |
| 子元素选择器 | `div>p`    | 选择div的直接子元素p     |
| 相邻兄弟     | `h1+p`     | 选择紧接在h1后的p元素    |
| 属性选择器   | `[target]` | 选择带有target属性的元素 |

### 盒子模型
- **组成**：content(内容) + padding(内边距) + border(边框) + margin(外边距)
- **box-sizing**：
  - `content-box`：默认，width/height只包含内容
  - `border-box`：width/height包含内容+padding+border

### 常用CSS属性
```css
/* 文本 */
color: #333;
font-size: 16px;
font-family: Arial;
text-align: center;
line-height: 1.5;
text-decoration: none;

/* 背景 */
background-color: #fff;
background-image: url('image.jpg');
background-repeat: no-repeat;
background-position: center;

/* 盒子模型 */
width: 100px;
height: 100px;
padding: 10px;
margin: 20px;
border: 1px solid #000;
border-radius: 5px;

/* 布局 */
display: block/inline/inline-block/flex;
position: static/relative/absolute/fixed;
float: left/right;
clear: both;
```

### Flex布局
```css
.container {
  display: flex;
  flex-direction: row; /* 主轴方向: row/column */
  justify-content: center; /* 主轴对齐 */
  align-items: center; /* 交叉轴对齐 */
  flex-wrap: wrap; /* 换行 */
}

.item {
  flex: 1; /* 分配剩余空间 */
  order: 1; /* 排列顺序 */
  align-self: flex-start; /* 单个项目对齐 */
}
```

### 响应式设计
```css
/* 媒体查询 */
@media (max-width: 768px) {
  body {
    font-size: 14px;
  }
}

/* 视口单位 */
width: 100vw; /* 视口宽度 */
height: 100vh; /* 视口高度 */
font-size: 2rem; /* 根元素字体大小的倍数 */
```

## Part III JavaScript

### 变量声明
```javascript
let x = 10; // 可重新赋值
const PI = 3.14; // 不可重新赋值
var y = 5; // 旧方式，函数作用域
```

### 数据类型
- 基本类型：Number, String, Boolean, Null, Undefined, Symbol
- 引用类型：Object, Array, Function

### 运算符
```javascript
// 算术
+ - * / % ++ --

// 比较
== != === !== > < >= <=

// 逻辑
&& || !

// 三元
condition ? expr1 : expr2
```

### 控制结构
```javascript
// 条件
if (condition) {
  // ...
} else if (condition) {
  // ...
} else {
  // ...
}

// switch
switch(expr) {
  case value1:
    // ...
    break;
  default:
    // ...
}

// 循环
for (let i = 0; i < 10; i++) {
  // ...
}

while (condition) {
  // ...
}

do {
  // ...
} while (condition);
```

### 函数
```javascript
// 函数声明
function sum(a, b) {
  return a + b;
}

// 函数表达式
const sum = function(a, b) {
  return a + b;
};

// 箭头函数
const sum = (a, b) => a + b;

// 默认参数
function greet(name = 'Guest') {
  return `Hello ${name}`;
}
```

### 数组操作
```javascript
let arr = [1, 2, 3];

// 常用方法
arr.push(4); // 末尾添加
arr.pop(); // 移除最后一个
arr.shift(); // 移除第一个
arr.unshift(0); // 开头添加
arr.slice(1, 3); // 截取
arr.splice(1, 1, 'a'); // 删除并添加
arr.map(x => x * 2); // 映射新数组
arr.filter(x => x > 1); // 过滤
arr.reduce((acc, cur) => acc + cur, 0); // 累积
```

### 对象操作
```javascript
let obj = {
  name: 'John',
  age: 30
};

// 访问属性
obj.name; // 'John'
obj['age']; // 30

// 添加/修改属性
obj.job = 'Developer';

// 删除属性
delete obj.age;

// 遍历
for (let key in obj) {
  console.log(key, obj[key]);
}

// 对象解构
const { name, age } = obj;
```

### DOM操作
```javascript
// 选择元素
document.getElementById('id');
document.querySelector('.class');
document.querySelectorAll('p');

// 修改内容
element.textContent = '新文本';
element.innerHTML = '<strong>HTML</strong>';

// 修改样式
element.style.color = 'red';
element.classList.add('active');
element.classList.remove('active');
element.classList.toggle('active');

// 事件监听
element.addEventListener('click', function(e) {
  console.log('点击了', e.target);
});

// 创建元素
const newDiv = document.createElement('div');
document.body.appendChild(newDiv);

// 移除元素
element.parentNode.removeChild(element);
```

### 异步编程
```javascript
// 回调函数
function fetchData(callback) {
  setTimeout(() => {
    callback('数据');
  }, 1000);
}

// Promise
fetch('url')
  .then(response => response.json())
  .then(data => console.log(data))
  .catch(error => console.error(error));

// Async/Await
async function getData() {
  try {
    const response = await fetch('url');
    const data = await response.json();
    console.log(data);
  } catch (error) {
    console.error(error);
  }
}
```

### ES6+特性
- 模板字符串：`` `Hello ${name}` ``
- 解构赋值：`const [a, b] = [1, 2]`
